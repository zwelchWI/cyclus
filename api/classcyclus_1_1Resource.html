Detailed Description

Resource defines an abstract interface implemented by types that are offered, requested, and transferred between simulation agents.

Resources represent the lifeblood of a simulation.

void cyclus::Resource::BumpId ( )

Assigns a new, unique id to this resource and its state.

This should be called by resource implementations whenever their state changes. A call to BumpId is not necessarily accompanied by a change to the state id. This should NEVER be called by agents.
